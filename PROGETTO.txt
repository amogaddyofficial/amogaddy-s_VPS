1. DESCRIZIONE GENERALE
================================================================================

amogaddy's VPS è una piattaforma PaaS (Platform as a Service) cloud pensata
per sviluppatori e principianti italiani. Offre 4 categorie di prodotti:

  a) App & Bot Hosting — gratuito, finanziato da pubblicità
  b) VPS Linux — da gratis a pagamento, su Google Cloud e Scaleway
  c) VPS Mac — Apple Silicon condiviso via SSH (in arrivo)
  d) Piano PAAS — Scaleway headless pay-per-second tramite Wallet interno

Il design segue il "Tema Giove": dark mode profondo (spazio scuro #0a0a0f),
accenti arancio/ambra (#f0a04b), effetti glassmorphism, font bold italic.

Obiettivo principale: offrire il primo accesso al cloud completamente gratis,
con monetizzazione tramite upgrade a piani a pagamento e pubblicità.

2. ARCHITETTURA TECNICA
================================================================================

FRONTEND
  - React 19 + Vite 8
  - Tailwind CSS 3 (palette personalizzata "Jupiter" + "Space")
  - Framer Motion (animazioni)
  - Lucide React (icone)
  - React Router DOM 7 (hash routing per GitHub Pages)

HOSTING FRONTEND
  - Test/staging: GitHub Pages (gratuito)
    URL: https://amogaddyofficial.github.io/amogaddy-s_VPS/
    Build: GitHub Actions con Node 22 + npm ci + vite build
  - Produzione definitiva: Vercel

BACKEND (fase futura - non ancora implementato)
  - Node.js / Express su Render
  - Database: PostgreSQL
  - Cronjob: reset tempo gratuito, gestione Caveau
  - API Scaleway: creazione/gestione istanze VPS
  - Webhook PayPal Standard per ricariche Wallet

INFRASTRUTTURA VPS UTENTI
  - Google Cloud Sandbox: piani FREE e FREE ADVANCED, App Hosting
  - Scaleway: piani HOME, PAAS, Mac Home, Mac Dev

4. LOGICHE DI BUSINESS
================================================================================

POLITICA NO-REFUND
  Il credito inserito nel Wallet non è mai rimborsabile o convertibile in
  denaro reale. Eccezioni: guasti hardware gravi o downtime superiore a 48
  ore consecutive. L'utente accetta esplicitamente alla registrazione.

WALLET INTERNO
  - Ricarica minima: 1,50 €
  - Metodo: PayPal Standard (webhook)
  - Il credito viene scalato al secondo per il piano PAAS
  - Storico transazioni visibile nella dashboard

SISTEMA PRORATA (downgrade)
  Quando un utente effettua un downgrade di piano, il sistema calcola
  istantaneamente il credito residuo per i giorni non utilizzati e lo
  accredita nel Wallet interno (mai su PayPal).

CAVEAU DI EMERGENZA (App Hosting)
  Dopo 3 giorni di inattività l'app viene archiviata:
  - Il codice viene salvato in un Git archive sicuro
  - La pagina mostra un overlay con video sponsorizzato obbligatorio
  - Dopo 10 secondi di video l'utente può riattivare l'app
  - Zero costi aggiuntivi — il video finanzia il ripristino

RESET GIORNALIERO APP HOSTING
  Ogni giorno alle 12:00 ora italiana l'ambiente dell'app viene riciclato
  e rilanciato automaticamente. I file nel File Manager rimangono intatti.

LISTA D'ATTESA MAC
  I piani Mac non sono ancora disponibili. Gli utenti possono registrare
  la loro email per ricevere accesso prioritario. Posizione in lista
  mostrata in tempo reale nella dashboard.

VINCOLO LEGALE REGISTRAZIONE
  L'utente non può completare la registrazione senza spuntare:
  "Accetto i Termini di Servizio e la Politica No-Refund Rigida"

6. ROUTING E NAVIGAZIONE
================================================================================

Usa React Router DOM v7 con createHashRouter (compatibile GitHub Pages).

Rotte pubbliche (senza login):
  /#/            → LandingPage
  /#/docs        → DocsPage (documentazione pubblica)
  /#/auth        → AuthPage (login/registrazione)

Rotte dashboard (post-login):
  /#/dashboard/home          → HomeDashboard
  /#/dashboard/app-hosting   → AppHosting
  /#/dashboard/vps           → VPSLinux
  /#/dashboard/mac           → MacPlans
  /#/dashboard/wallet        → WalletPage
  /#/dashboard/wiki          → Wiki

Link documentazione: accessibile dalla navbar e footer della landing.

  Step 2 — Nome istanza + slider risorse
    Slider CPU (%): da 10% al limite del piano
    Slider RAM (MB): da 128MB al limite del piano
    Slider Disco (GB): da 1GB al limite del piano
    Limiti per piano:
      FREE:       CPU 30%,  RAM 512MB,  Disco 5GB
      FREE ADV:   CPU 60%,  RAM 1024MB, Disco 10GB
      HOME:       CPU 100%, RAM 4096MB, Disco 25GB
      PAAS:       CPU 100%, RAM 8192MB, Disco 50GB
  Step 3 — Riepilogo completo prima della conferma

APPHOSTINGPAGE (src/pages/AppHosting.jsx)
  Stato A (Creazione):
    Selezione runtime: Python / Node.js / Go / HTML Statico

  Stato B (Attivo):
